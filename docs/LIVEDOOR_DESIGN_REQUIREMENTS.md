# livedoorフロント表示要件

## PCトップページ

- 現在の記事ページと同じ黒背景・ピンク見出し・白い画像カードのデザイン
- 1回の自動投稿につきトップページへ1カード追加
- 横5列 × 縦8行 = 1ページ40投稿
- 41件目以降はlivedoor側のページ送りで次ページへ表示

## PC記事ページ

- 現在と同じ画像カードデザイン
- 横5列 × 縦10行 = 最大50リンク
- 各カードは画像・サイト名・記事タイトルを表示
- カード全体を元サイトの記事へのリンクにする
- ページ下部のコメント欄を非表示

## 注意

トップページの40件表示、ページ送り、コメント欄の削除はlivedoorブログ側のPC用HTMLテンプレートとCSSの編集が必要です。このPHPツールからAtomPub APIだけでlivedoorテーマ自体を変更することはできません。

正確なテンプレート改修には、現在livedoorで使用中の以下を確認して編集します。

- PC用トップページHTML
- PC用個別記事ページHTML
- PC用CSS

Markdownのコード枠（```html と ```）はテンプレートへ貼り付けません。
