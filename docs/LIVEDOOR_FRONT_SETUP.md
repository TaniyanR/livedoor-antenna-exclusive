# livedoorブログ側の表示設定

投稿本文は、画像カードのレイアウト・文字色・余白をすべてインラインスタイルで出力します。livedoor側へ専用CSSを貼り付ける必要はありません。

## 現在の崩れを直す手順

1. livedoor Blogの「ブログ設定」→「デザイン／ブログパーツ設定」→「PC」を開きます。
2. 現在のカスタムデザインを保存してバックアップします。
3. HTML内に表示されている ` ```html ` と、末尾の ` ``` ` を削除します。
4. トップページで記事本文を表示しない独自テンプレートを使用している場合は、いったんlivedoor標準デザインへ戻します。
5. 新しい投稿を1件テストし、画像カードが表示されることを確認します。

## 新しい投稿本文の表示

- PCでは画像カードが横に複数並びます。
- 画面幅が狭くなると自動的に折り返します。
- 画像は固定枠内で縦横比を保ち、切り抜かずに表示します。
- 画像がない記事には「画像なし」を表示します。
- カード全体が元記事へのリンクです。
- カードにはRSSサイト名と記事タイトルを表示します。
- 黒背景のテーマでも、カード内は白背景・黒文字で読めるようにしています。

## 注意

修正前に投稿済みの記事本文は自動では書き換わりません。修正後に作成した新しい投稿から新レイアウトが適用されます。
