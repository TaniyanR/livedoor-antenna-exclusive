livedoorアンテナ

RSS aggregation and automated posting system for livedoor blog antenna sites.

livedoorアンテナは、複数のRSSフィードを収集し、記事を自動集約してlivedoorブログへ投稿するためのアンテナサイト運用システムです。

取得記事の管理、URL重複除外、自動投稿、Cron運用、RSS配信までを一元管理できます。

共有サーバー環境での運用を前提とし、シンプルかつ実運用重視で設計されています。

⸻

主な機能

RSS管理

* RSS登録
* RSS編集
* RSS削除
* RSS接続確認
* RSSテスト取得
* RSSプレビュー（3件）
* RSS検索
* ドラッグ＆ドロップ並び替え
* 最終取得結果表示
* 最終取得日時表示

登録RSSはすべて取得対象です。

ON/OFF機能は実装しません。

⸻

RSS取得

対応形式

* RSS
* Atom

取得機能

* URL重複除外
* タイトル重複許可
* 記事DB保存
* 複数RSS対応
* RSSごとの取得状況管理

画像取得優先順位

1. media:content
2. media:thumbnail
3. enclosure
4. description内img
5. content内img

画像本体は保存しません。

画像URLのみ保存します。

⸻

記事管理

取得した記事を管理できます。

機能

* カード表示
* 画像表示
* 記事タイトル表示
* カード全体クリック
* タイトル検索
* キーワード検索
* ソート
* 個別削除
* 一括削除
* ページネーション

削除はDB上のみ行われます。

既に投稿済みのlivedoor記事は変更されません。

⸻

livedoorブログ自動投稿

AtomPub APIを利用してlivedoorブログへ投稿します。

機能

* 自動投稿
* 手動投稿
* 即公開投稿
* 投稿履歴保存
* 接続テスト

投稿対象記事は、

* 新着順ベース
* 軽い偏り補正

で抽出されます。

同一RSSへの偏りを抑えながら、自然なアンテナサイト運用を実現します。

⸻

投稿タイトル生成

投稿対象記事からメイン記事を選択し、自動でタイトルを生成します。

仕様

* 偏り付きランダム抽出
* 画像あり記事優遇
* タイトル整形
* 不要なサイト名除去
* 不要な接頭辞除去

⸻

Cron対応

対応機能

* RSS取得Cron
* 投稿Cron
* ログ記録
* 自動削除処理

管理画面から、

* 実行間隔設定
* Cron設定例確認

が可能です。

シンサーバー環境を想定しています。

⸻

RSS配信

ツール自身のRSSを配信します。

URL

/rss.xml

仕様

* RSS2.0
* 最新20件
* 軽量description
* 画像タグ対応

RSSリンク先はlivedoor投稿記事です。

管理機能

* 一括削除

* 最終RSS取得結果

管理機能

* RSS管理

インストーラー

初回アクセス時にセットアップを行います。

入力項目

* DBホスト
* DB名
* DBユーザー
* DBパスワード
* 管理者ユーザー名
* 管理者パスワード
* サイト名
* サイト説明

セットアップ内容

* DB接続確認
* テーブル作成
* 設定ファイル生成
* 初期管理者作成

⸻

基本設定

管理画面から変更可能

* サイト名
* サイト説明
* RSS取得件数
* 投稿件数
* 投稿間隔
* 投稿履歴保持期間

初期値

* RSS取得件数：20
* 投稿件数：36
* RSS配信件数：20
* 投稿履歴保持期間：90日

⸻

livedoor設定

管理画面から設定可能

* ブログID
* AtomPub URL
* livedoor ID
* AtomPubパスワード
* 接続テスト

API設定をコード編集なしで管理できます。
